O Spring Boot é um framework que torna fácil criar aplicações Spring que chegam prontas para rodar, necessitando de configuração mínima. O Spring Boot já vem com um servidor de aplicação embarcado por padrão, o Apache Tomcat, configura automaticamente o Spring e algumas bibliotecas, não requer configuração XML e já vem pronto para rodar. A adição de novas dependências é feita pelo Apache Maven, através do arquivo pom.xml, mas as dependências podem também ser gerenciadas pelo Gradle caso o projeto tenha sido configurado para utilizá-lo. Na minha opinião a melhor IDE para trabalhar com o Spring Boot é o IntelliJ IDEA, por ter sido a primeira IDE que eu utilizei quando comecei a aprender Java. O Spring Initialzr é uma ferramenta web utilizada para gerar a estrutura de projetos Spring Boot.
